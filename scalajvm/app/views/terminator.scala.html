The SBOL symbol for a terminator is a T shape.

Rightwards: represents a terminator on the forward/top strand.
Leftwards: represents a terminator on the backwards/reverse-complement/bottom strand.

The symbol for a terminator has a variable width and depth.

The terminator symbol can be aligned to the backbone in three modes: backbone-centred, above-strand and below-strand. The strand-centred terminator is centred on the backbone line. For above-strand and below-strand, it is always placed that side. The outside label is always placed furthest from the strand.

You can embed terminator symbols directly into the flow of text by using SBOLv the terminator shortcode [term]fish[/term] within normal text. If you don't need a label you can either use an empty shortcode ([term][/term]) or a self-closing shortcode ([term/]). The width and height of the symbol can be set manually as in [term width="100"]looong[/term] and [term width="20"/] and the orientation with [term dir="<"/] and [term dir=">"/].
